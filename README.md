Hostel Exchange

A secure hostel-only marketplace where students can buy, sell, rent, or request items within their community. Listings are verified, prices stay below MRP, and temporary ownership tracking ensures safe rentals, trusted exchanges, smooth transactions, and reliable student-to-student sharing.

Problem

Hostel students frequently face problems such as:

Difficulty selling items when leaving the hostel

No trusted platform to buy second-hand items nearby

Temporary need for items like calculators, irons, tools, or cycles

Risk of fraud when using public marketplaces

Overpriced second-hand products

Existing platforms like OLX are not designed for closed campus communities.

Solution

Hostel Exchange is a hostel-specific marketplace that allows students to safely exchange items within their community.

The platform enables:

Buying and selling items

Renting items temporarily

Requesting unavailable items

Tracking temporary ownership for rentals

Fair pricing using MRP verification

Admin moderation for safety

Key Features
Marketplace

Students can list items for:

Sale

Rent

Listings include:

Item name

Category

Condition

Images

Price

MRP

Hostel location

Rental System

Items can be rented with defined terms:

rent price per day

maximum duration

deposit amount

late return penalty

Temporary Ownership Transfer

When an item is rented, the app records a temporary ownership transfer.

This tracks:

original owner

temporary holder

start time

return deadline

deposit amount

This prevents disputes and ensures responsible usage.

Item Request System

If an item is unavailable, users can request it.

Example request:

Item: Scientific Calculator
Max Price: ₹400
Urgency: High
Request Scoring

Requests are scored based on:

number of requests

urgency

historical demand

High-scoring requests trigger notifications to potential sellers.

Seller Notifications

Users who may own a requested item receive alerts.

Example:

12 students requested Scientific Calculators.
Do you want to list yours?

This helps balance demand and supply automatically.

Fair Pricing (MRP Check)

To prevent overpricing:

Selling Price ≤ MRP

Listings that exceed the MRP are rejected.

Restricted Item Detection

The system prevents listing of:

adult items

alcohol

weapons

illegal products

AI image moderation tools such as
Google Cloud Vision API can assist with detection.

Messaging System

Buyers and sellers can communicate through in-app chat to:

ask questions

negotiate prices

coordinate pickup

Reviews & Trust Score

Users can rate each other after transactions.

Example:

⭐⭐⭐⭐⭐
Item was exactly as described.

Trust score improves platform reliability.

Lost & Found

Students can report:

lost items

found items

Example:

Lost: Black wallet near mess
Found: USB drive in library
Payments

Supported payment methods:

Cash

Google Pay

PhonePe
